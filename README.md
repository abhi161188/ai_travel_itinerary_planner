Tech Stack:
1. Groq --> LLM
2. GCP VM --> To deploy and run you app on the cloud within a Virtual Machine. It's a service offfered by Google Cloud.
3. Langchain --> Generative AI Framework to interact with LLM.
4. Minikube --> Making a Kubernetes Cluster inside a VM.
5. Docker --> For containerization of the app during deployment.
6. Stremlit --> To make UI or frontend of the app.
7. Kubectl --> CLI tool for Kubernete to interact with Minikube.
8. GitHub --> It will work as a SCM for your project.
9. FileBeat --> Read logs from your Kubernete Servers/Applications and forwards them.
10. Logstash --> Process logs from Filebeat like cleaning of logs, adding tags, etc before sending them to storage.
11. ElasticSearch --> Stores logs in a searchable format so that we can easily access logs.
12. Kibana --> Provides you dashboard services to beautifully view your logs.
